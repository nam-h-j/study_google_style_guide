- google code style guide 기록 및 정리

## 스타일 가이드가 필요한 이유
- 코드는 쓰는(코딩) 횟수보다 읽는(분석) 횟수가 훨씬 많다.
- 그래서 많은 시간적 비용은 읽을 때 발생한다.
- 이 비용을 줄이기 위해서 가독성을 최대한 높여야 한다.
- 그렇기 때문에 스타일 가이드에 맞춰 작성하는 것은 중요하다.
